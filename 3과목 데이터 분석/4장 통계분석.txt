* 정상성 
	1) 시계열의 평균과 분산에 체계적인 변화 및 주기적 변동이 없다는 것
	2) 미래는 확률적으로 과거와 동일하다는 것

* 정상성 시계열의 조건 
	1) 평균은 모든 시점(시간 t)에 대해 일정하다. E(x) = (u)평균
	2) 분산은 모든 시점(시간 t)에 대해 일정하다. Var(x의t) = a의 2곱(분산)
	3) 공분산은 시점(시간 t)에 의존하지 않고(관계X), 단지 시차에만 의존한다(관계O). Cov(x(i+h), x(t)) = Y(h)

* 정상 시계열 전환
* 정상 시계열로 전환하는 방법 
	1) 비정상시계열 자료는 정상성을 만족하도록 데이터를 정상시계열로 만든 후 시계열 분석을 수행한다.
	2) 평균이 일정하지 않은 경우 : 원계열에 "차분" 사용
	3) 계절성을 갖는 비정상시계열 : "계절차분" 사용
	4) 분산이 일정하지 않은 경우 : 원계열에 "자연로그(변환)" 사용

* 차분 : 현 시점의 자료 값에서 전 시점의 자료 값을 빼 주는 것 의미함

* 시계열 모형
* AR 모형 자기회귀모형 (정상성 시계열)
	1) AR(p) : 현 시전의 자료를 p 시점 전의 유한 개의 자기 자신의 과거 값을 사용하여 설명
	2) 백색 잡음의 현재 값과 자기 자신의 과거 값의 선형 가중 값으로 이루어진 정상 확률 모형
	3) 현 시점의 시계열 자료에 과거 1시점 이전의 자료만 영향을 주면 이를 1차 AR 모형이라하고 AR(1) 라고 표기함

* MA 모형 이동평균 모형 (정상성 시계열)
	1) MA(q) : 과거 q 시점 이전 오차들에서 현재항의 상태를 추론한다.
	2) 최근 데이터의 평균을 예측치로 사용하는 방법, 각 과거치는 동일 가중치가 주어짐
	3) 현시점의 자료가 유한 개의 백색잡음의 선형결합으로 표현되었기 때문에 항상 정상성을 만족함

* ARIMA 모형 자기회귀 누적 이동평군 모형 (비정상성 시계열)
	1) 현재와 추세간의 관계를 정의한 것, 많은 시계열 자료가 ARIMA 모형을 따름
	2) ARIMA 모형은 비정상시계열 모형이며, 차분/변환을 통해 AR, MA, ARMA 모형으로 정상화 할 수 있음
	3) ARIMA(p, d, q) -> p : AR모형 차수, d : 차분, q : MA모형 차수
		- ARIMA(1, 2, 3) : 이라면 2번 차분해서 ARMA 모형이 될 수 있음
		- ARIMA(0, 1, 3) : IMA(1, 3) 모형이고 이것을 1번 차분하면 MA(3) 모형이됨
		- ARIMA(2, 3, 1) : ARI(2, 3) 모형이고 이것을 3번 차분하면 AR(2) 모형이 됨

* ACF, PACF, White Noise
* ACF, 자기 상관 함수
	1) Auto-Correlation Function, 시계열 데이터의 자기상관성을 파악하기 위한 함수
	2) 시계열의 관측치 Y(t)와 Y(t-k) 간 상관계수를 k의 함수 형태로 표기한 것, k : 시간단위
	3) -1 <= autocorr(Y(t), Y(t-k)) <= 1, k가 커질수록 ACF는 0으로 수렵함

* PACF(Partial ACF)
	1) Y(t)와 Y(t-k) 중간에 있는 값들의 영향을 제외시킨 Y(t)와 Y(t-k) 사이의 직접적 상관관계를 파악하기 위한 함수

>>> 특강7. [ 카이제곱검정, 회귀모형의 ANOVA ] 9:41
* 카이제곱 검정
	- 범주형 자료로 구성된 데이터 분석에 사용
	- 관찰된 빈도가 기대되는 빈도와 유의한 차이가 있는지 검증
	- 카이제곱값 = Σ(관측값 - 기댓값)^2 / 기댓값

* 카에제곱 검정 중 독립성 검정
	- Cotingency table에 있는 두 개 이상의 변수가 서로 독립인지 검정
	- 귀무가설 : 두 변수는 차이가 없음 ( 독립O, 관계X ) ; 연체와 학생은 서로 독립니다.
	- 독립가설 : 두 변수는 차이가 있음 ( 독립X, 관계O ) ; 연체와 학생은 서로 독립이 아니다.

* 회귀모형의 anova : 회귀 모형의 유의성 분석
	- 관측치 : df + k(독립변수 개수) + 1 = 48 + 1 + 1 = 50
	- 결정계수(R2 score) = SSR/SST(SSR+SSE) = ((21186)/(21186+11354)) = 0.651
	- 오차 분산의 불편추정량은 'MSE' 오차제곱평균으로, Mean Sq와 Residuals가 교차되는 지점에 236.5이라고 써 있음
